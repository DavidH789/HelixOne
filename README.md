# 🧬 HelixOne

**Особистий реєстр походження — зведення ДНК-етнічних звітів з кількох джерел в одному місці.**

HelixOne допомагає звести відсоткові результати з різних ДНК-лабораторій (GEDmatch, MyHeritage, FTDNA, 23andMe тощо) в одну картину: усереднення, медіана, довіра між джерелами, конфлікти, кластери споріднених груп.

> Ніяких нових наукових обчислень. HelixOne нічого не вигадує — лише чесно усереднює й зіставляє те, що вже порахували лабораторії.

## ✨ Можливості

- **Автопарсинг звітів** — вставляєш скопійований текст звіту, скрипт сам розпізнає назви груп і відсотки
- **Імпорт файлів** `.csv`, `.txt`, `.json` (включно з повним бекапом самого HelixOne)
- **Усереднення з кількох джерел** — середнє, медіана, std, min–max, індекс узгодженості
- **Довіра ★** — залежить від кількості незалежних джерел і того, наскільки вони узгоджені
- **Ручні кластери** — об'єднання споріднених груп (напр. "Mediterranean") з автопідказками
- **Версії (W1, W2…)** — знімки стану з порівнянням будь-яких двох версій
- **"Відомі пращури"** — документальні знахідки поруч із ДНК-групами, без автоматичних вердиктів
- **Збіги з родичами (сМ)** — з фільтром і сортуванням за національністю
- **Кругова діаграма й графіки** — власна SVG-реалізація, без зовнішніх бібліотек
- **Експорт** — JSON (повний бекап), CSV, друкований "Генетичний паспорт" (PDF), PNG-картка для соцмереж
- **Архів замість видалення**, **скасувати останню дію**, **нагадування про бекап**
- **Простий/Науковий режим** перемикання глибини деталей

## 🚀 Як користуватись

Жодного встановлення не потрібно. HelixOne — це один самодостатній HTML-файл.

1. Завантаж [`index.html`](./index.html)
2. Відкрий його в будь-якому браузері (Chrome, Safari, Firefox)
3. Все

Дані зберігаються локально у твоєму браузері (`localStorage`) — нікуди не відправляються.

### Онлайн-демо

Можна відкрити прямо через GitHub Pages (якщо увімкнено для цього репозиторію) — без завантаження файлу.

## 🛠 Технічно

- Чистий JavaScript (без React/Vue/фреймворків)
- Нуль зовнішніх залежностей під час виконання (тільки Google Fonts)
- Усі іконки, графіки та QR/PNG-експорт — власні реалізації на SVG/Canvas
- Один файл — легко відкрити будь-де, включно з обмеженими мобільними переглядачами

## ⚠️ Важливо

HelixOne — це інструмент для особистого зведення й організації вже готових результатів. Це **не** заміна професійної генетичної консультації, і він не виконує власного ДНК-аналізу сирих даних.
